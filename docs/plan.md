# TECC – Planejamento
## Objetivo do projeto

Criar um site que explique teorias científicas complexas de forma simples utilizando explicações visuais e interatividade.

## Público-alvo

Crianças entre 10 e 14 anos interessadas em ciência.

## Estrutura das teorias
Cada teoria terá:
- Explicação simples
- Exemplo prático ou experimento interativo
- Curiosidade científica
- Material complementar (vídeos educativos)

## Categorias
- Física e astronomia
- Biologia
- Tecnologia
- Matemática

## Páginas do site
- Página inicial
- Catálogo de teorias com filtros por categoria
- Página de teoria
- Na página da teoria existirão seções como:

1. explicação
2. experimento interativo
3. curiosidades
4. materiais complementares
