# TECC – Produto

## Visão geral
TECC é um site educacional que explica teorias científicas complexas de forma simples para crianças curiosas. O objetivo é tornar conceitos difíceis mais acessíveis usando linguagem simples, exemplos do cotidiano e experimentos interativos.

A plataforma combina explicações visuais com pequenas simulações ou experiências digitais que ajudam o usuário a entender o conceito de forma prática.

## Público-alvo

- Crianças entre 10 e 14 anos interessadas em ciência e tecnologia
- Professores que desejam recursos visuais para explicar conceitos
- Pais que querem incentivar curiosidade científica

## Problema

Muitos conceitos científicos são ensinados de forma muito abstrata ou matemática, o que dificulta o interesse inicial das crianças.

## Solução

O TECC apresenta teorias com:

- linguagem simples
- exemplos do cotidiano
- curiosidades científicas
- experimentos interativos com imagens, animações ou simulações
